# Programma van Eisen \(PvE\)


**M**o**SC**o**W**
* `Must` (Moet er in)
* `Should` (Zou er in moeten, maar het kan ook zonder)
* `Could` have (Zou er in kunnen, maar niet heel belangrijk) 
* `Won't` have (Komt er niet in)

<br>
